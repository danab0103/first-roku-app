My first Roku App

How to run
1. Zip the contents of the hello-world directory.
2. In your web browser, enter the URL of your Roku device, and then log in (the User Name is "rokudev"; the password is the one you created when you activated developer mode on your Roku device.
3. In the Development Application Installer, click Upload, and then select the /hello-world-master/dist/apps/hello-world/*.zip file.
4. The app launches on your Roku device.
